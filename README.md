Funções (ou métodos) são blocos de código que realizam tarefas específicas, podendo receber parâmetros e retornar valores, permitindo a organização e reutilização do código.

Conteúdo dos exercícios

Criação de métodos simples: sem parâmetros e sem retorno

Métodos com parâmetros: passando valores para processamento

Métodos com retorno: retornando valores calculados

Sobrecarga de métodos: criar métodos com o mesmo nome, mas diferentes assinaturas

Métodos estáticos vs não estáticos: quando usar cada um

Exemplo:

Função somar(int a, int b) que retorna a soma de dois números

Função ehPar(int numero) que retorna true se o número for par

Tecnologias e Ferramentas

Java 17+

IDEs: IntelliJ
